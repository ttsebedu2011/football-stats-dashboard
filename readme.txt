Project Title: Interactive Football Statistics Dashboard

This project presents a web-based visualisation dashboard for exploring football statistics across multiple seasons, 
including the 2020/2021 Premier League. It integrates data from various CSV files to showcase team performances, 
player statistics, and league standings through dynamic and interactive charts and graphs. 
The dashboard allows users to interactively filter, analyse, and compare data, 
providing a comprehensive and insightful look into football trends and performances.

Data Source:
The data was sourced from the official Premier League websites https://footystats.org/download-stats-csv , https://www.kaggle.com/datasets/hugomathien/soccer  and is used under the terms of their data usage policy.

Setup Instructions:
1. Ensure you have a modern web browser installed (Google Chrome, Firefox, Microsoft Edge).
2. Download the project files from https://github.com/ttsebedu2011/football-stats-dashboard and save them to your local machine.
3. No additional dependencies are required.

Running the Project:
1. Navigate to the project directory where the 'index.html' file is located.
2. Open 'index.html' with your web browser to access the visualisation dashboard.
3. Use the interactive features by clicking on chart elements to filter and reveal more data.

Navigating the Dashboard:

- The dashboard is designed to be intuitive. Start by selecting a chart and use the filter options to refine the data displayed.
- Hover over data points, bars, and other elements to view detailed tooltips and additional data specifics.

Troubleshooting:
- If the charts do not load, check to ensure you're not blocking JavaScript execution in your browser settings.
- Make sure all files from the project folder are kept in the same directory to ensure proper linkage.
- Display Issues, should check for any browser-specific issues by opening the dashboard in a different browser.
